Object Detection Web Server

Welcome to the parasite object detection web server. This web server seeks to be the tool you need to detect the presence of parasites in human stool samples. This page is designed to help pharmacists, pharmacy students and people interested in the analysis of this type of samples.

The parasites that can be deteceted are:

  • 1.Trichuris trichura
  • 2. Ascaris lumbricoides
  • 3. Uncinarias
  • 4. Diphyllobotrium latum
  • 5. Taenia
  • 6. Balantidium coli
  • 7. Hymenolepis nana
  • 8. Enterobius vermicularis
  • 9. Amebas
  • 10. Giardia

    • The usage of the server is simple, but if you are a new user you can follow this steps:

  • 1. Press "Examinar..." and select the image of the sample that you want to analyze. This image must be an .jpg image!
  • 2. Press "Upload File". This will make the detection over your image, returning the new image with the detection done.
  • 3. If you want to save the image, press RIGHT+CLICK and save as.
